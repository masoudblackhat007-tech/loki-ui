````markdown
# بخش ۲۰ — اضافه شدن نمایشگر داخلی داکیومنت‌های دو‌زبانه

## هدف بخش

در این بخش برای پروژه `loki-ui` یک نمایشگر داخلی برای خواندن داکیومنت‌های پروژه ساخته شد.

هدف این بود که مستندات پیشرفت پروژه فقط به صورت فایل‌های جداگانه داخل repository باقی نمانند، بلکه از داخل خود UI قابل خواندن باشند.

این قابلیت برای داکیومنت‌سازی رزومه‌محور اضافه شد تا بخش‌های مختلف پروژه به شکل صفحه‌به‌صفحه، قابل مطالعه، مرتب و قابل ارائه باشند.

## محدوده تغییرات

تغییرات این بخش فقط مربوط به خود پروژه `loki-ui` بود.

در این بخش این قسمت‌ها تغییر کردند:

```text

templates/docs.tmpl

```

در این بخش این قسمت‌ها تغییر نکردند:

```text

این بخش یک تغییر در observability pipeline نبود.

این بخش یک قابلیت UI و documentation viewer داخلی برای خواندن مستندات پروژه بود.

## مدل دسترسی

مدل دسترسی `loki-ui` تغییر نکرد.

برنامه همچنان فقط روی loopback سرور گوش می‌دهد:

```text
127.0.0.1:18090
```

مدل دسترسی همچنان این است:

```text
browser -> SSH tunnel -> 127.0.0.1:18090 on server -> loki-ui
```

پورت `18090` نباید عمومی شود.

این بخش نباید به عنوان public documentation portal معرفی شود.

این بخش فقط یک internal documentation viewer داخل همان UI داخلی است.

## وضعیت قبل از شروع

برای خواندن داک‌ها باید فایل‌ها مستقیم باز می‌شدند و تجربه خواندن داخل UI وجود نداشت.

همچنین ساختار رسمی برای نسخه فارسی و انگلیسی وجود نداشت.

## رفتار انتخاب زبان

زبان از query parameter زیر خوانده می‌شود:

```text
lang
```

اگر مقدار `lang` برابر `en` باشد، داک انگلیسی نمایش داده می‌شود.

اگر مقدار `lang` خالی یا نامعتبر باشد، زبان پیش‌فرض فارسی است.

زبان فارسی با مقدار زیر مشخص می‌شود:

```

زبان انگلیسی با مقدار زیر مشخص می‌شود:

```

## جهت متن

برای انگلیسی، جهت متن چپ‌به‌راست تنظیم شد:

برای همین، جهت متن بخشی از طراحی اصلی این feature بود.

## fallback فارسی

برای جلوگیری از شکستن حالت فارسی در دوره انتقال، اگر مسیر فارسی جدید خالی باشد، handler می‌تواند فایل‌های قدیمی داخل مسیر اصلی را بخواند.

این fallback فقط برای فارسی در نظر گرفته شد.

هدف fallback این بود که در زمان جابه‌جایی داک‌ها از مسیر قدیمی به مسیر جدید، صفحه داک کاملاً از کار نیفتد.

```text
````
